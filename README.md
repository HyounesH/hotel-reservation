
# Hotel Reservation Flutter App

This project is a Flutter application focused on the design of a modern hotel reservation experience. The app features a clean, intuitive interface for exploring hotels and selecting reservation dates.

## Getting Started

To clone and set up this project:

1. **Clone the repository:**
	```sh
	git clone <repository-url>
	cd hotel_reservation
	```

2. **Get dependencies:**
	```sh
	flutter pub get
	```

3. **(Optional) Update dependencies:**
	```sh
	flutter pub upgrade
	```

4. **Run the app:**
	```sh
	flutter run
	```

Replace `<repository-url>` with the actual URL of your repository.

## Design Overview

### Explorer Interface
The explorer interface allows users to browse available hotels, view details, and navigate through featured listings. The design emphasizes usability and visual appeal, with prominent images and easy-to-read information.

![Explorer Interface](images/explorer%20interface.png)

### Calendar Interface
The calendar interface provides a user-friendly way to select check-in and check-out dates. It uses a visually distinct calendar component, making date selection straightforward and visually engaging.
